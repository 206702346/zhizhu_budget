# 小账本 — 个人记账微信小程序

> 基于 **微信小程序 + 云开发** 的全功能个人记账应用，支持收支记录、预算管理、图表统计和数据导出。

---

## 📱 项目概览

**小账本** 是一款面向个人日常财务管理的微信小程序。用户可以快速记录每一笔收入和支出，按月份查看账单明细，通过可视化图表分析消费结构，设置月度预算控制开销，并导出账单数据或分享给好友。

## ✨ 功能清单

### 🔐 用户系统
- 微信一键登录，自动获取 `openid`
- 基于 `_openid` 的多用户数据隔离，每位用户仅可操作自己的账单

### 📝 账单管理
- **新增账单** — 选择收入/支出类型，输入金额、分类、日期和备注
- **编辑账单** — 点击账单进入编辑弹窗，修改任意字段
- **删除账单** — 确认弹窗后安全删除
- **月份筛选** — 按月份快速切换查看

### 📊 数据统计
- **本月概览** — 收入、支出、结余、记录笔数一目了然
- **环形饼图** — Canvas 2D 自绘，直观展示各支出分类占比
- **柱状趋势图** — 近 6 个月收支对比，消费趋势清晰可见
- **分类排行** — 本月支出分类金额排序 + 彩色标识

### 💰 预算管理
- 自定义月度支出预算金额
- 实时进度条展示已花费比例
- 超支自动红色警告提示

### 🔍 搜索筛选
- 按**关键词**搜索（匹配备注和分类名称）
- 按**金额范围**筛选
- 搜索结果即时汇总统计

### 📂 分类管理
- 收入/支出分类分别自定义
- 支持添加、删除和排序
- 可一键恢复默认分类

### 📤 数据导出
- Canvas 2D 渲染精美统计报告图片
- 一键保存到手机相册

### 🔗 微信分享
- 4 个页面均支持分享给好友和朋友圈
- 分享文案自动带入当月统计数据

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────┐
│              微信小程序（前端）                │
│  WXML / WXSS / JavaScript                   │
│  4 个 Tab 页面：首页 / 记一笔 / 账单 / 我的     │
│  Canvas 2D 图表自绘（零第三方依赖）              │
└──────────────────┬──────────────────────────┘
                   │  wx.cloud.callFunction
                   ↓
┌─────────────────────────────────────────────┐
│              云函数（后端）                     │
│  quickstartFunctions                        │
│  Node.js + wx-server-sdk                   │
│  10+ 个业务函数                              │
└──────────────────┬──────────────────────────┘
                   │
                   ↓
┌─────────────────────────────────────────────┐
│              云数据库                         │
│  bills      — 账单记录                       │
│  budgets    — 月度预算                       │
│  categories — 自定义分类                     │
└─────────────────────────────────────────────┘
```

### 技术选型

| 层 | 技术 | 说明 |
|---|---|---|
| 视图层 | WXML + WXSS | 微信小程序原生框架 |
| 逻辑层 | JavaScript (ES6+) | async/await 异步处理 |
| 后端 | 云函数 | Node.js，`wx-server-sdk ~2.4.0` |
| 数据库 | 云数据库 | JSON 文档型，自动索引 |
| 图表 | Canvas 2D API | 自绘环形饼图 + 柱状图，无第三方依赖 |
| 导出 | Canvas 离屏渲染 | 渲染统计报告 → 保存相册 |

## ☁️ 云函数接口

所有业务逻辑通过云函数 `quickstartFunctions` 处理，以 `event.type` 区分操作：

| type | 功能 | 说明 |
|---|---|---|
| `getOpenId` | 获取用户 openid | 登录认证 |
| `addBill` | 新增账单 | 自动注入 _openid |
| `getBills` | 查询账单 | 支持按月、按类型、分页 |
| `updateBill` | 更新账单 | 仅可修改本人账单 |
| `deleteBill` | 删除账单 | 仅可删除本人账单 |
| `getMonthStats` | 月度统计 | 收入/支出/结余/分类汇总 |
| `getOverviewStats` | 总览统计 | 总账单数 + 本月汇总 |
| `searchBills` | 搜索账单 | 关键词 + 金额范围 |
| `getCategories` | 获取分类 | 用户自定义分类列表 |
| `saveCategories` | 保存分类 | 全量替换 |
| `getBudget` | 获取预算 | 按月份查询 |
| `setBudget` | 设置预算 | 新增或更新 |

## 🚀 部署指南

### 环境要求

- 微信开发者工具（最新版）
- 已注册微信小程序，并开通云开发

### 部署步骤

**1. 克隆项目**

将项目文件夹导入微信开发者工具。

**2. 配置云环境**

修改 `miniprogram/envList.js` 中的 `envId`，或者在 `app.js` 中直接填入云环境 ID。

**3. 上传云函数**

在开发者工具左侧文件树中，右键 `cloudfunctions/quickstartFunctions` → **「上传并部署：云端安装依赖」**。

**4. 创建数据库集合**

在开发者工具顶部点击 **「云开发」** → **「数据库」**，创建以下 3 个集合：

- `bills`（账单）
- `budgets`（预算）
- `categories`（分类）

**5. 设置数据库权限**

每个集合的权限设置建议选择 **「仅创建者可读写」**。

**6. 编译运行**

点击开发者工具工具栏的 **「编译」**，即可在模拟器或真机预览中体验。

## 📖 使用说明

### 记一笔

1. 打开「记一笔」Tab
2. 选择 **支出** 或 **收入** 类型
3. 输入金额
4. 选择分类（可在「我的」页面自定义）
5. 选择日期（默认当天）
6. 可选填写备注
7. 点击 **「保存账单」**

### 查看账单

1. 打开「账单」Tab
2. 顶部切换月份查看历史记录
3. 点击 **「编辑」** 修改账单
4. 点击 **「删除」** 移除账单
5. 点击 **「搜索」** 按关键词或金额筛选
6. 点击 **「导出」** 保存统计图片到相册

### 设置预算

1. 打开「我的」Tab
2. 在「本月预算」卡片中点击 **「设置预算」**
3. 输入月度支出上限
4. 首页和我的页面将实时显示预算进度

### 管理分类

1. 打开「我的」Tab
2. 在「分类管理」卡片中点击 **「编辑」**
3. 添加 / 删除分类
4. 点击 **「保存」** 生效

## 🎯 课程设计参考

### 技术亮点（适合写入课设报告）

1. **三层架构** — 小程序前端 + 云函数后端 + 云数据库，职责清晰
2. **多用户隔离** — 基于 `openid` 的数据权限控制，保障数据安全
3. **自研图表** — Canvas 2D 原生 API 实现环形饼图和圆角柱状图，零第三方依赖
4. **异步编程** — 全链路使用 `async/await` + `Promise.all` 并行优化
5. **CRUD 完整闭环** — 数据的新增、查询（分页/筛选/搜索）、更新、删除
6. **图片导出** — Canvas 离屏渲染 + 相册保存
7. **响应式设计** — `rpx` 单位适配不同屏幕，深色现代 UI 风格
